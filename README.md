# AI Task Assistant

A full-stack productivity app that turns meeting notes into actionable tasks using AI. Paste your meeting notes, get a summary and prioritized task list generated by GPT-4o, and push tasks directly to your task board.

## Features

- **AI Meeting Summarizer** — paste meeting notes and get a concise summary + extracted action items via GPT-4o
- **Task Management** — create, update, and delete tasks with priority levels (low / medium / high)
- **One-click task import** — add AI-suggested tasks from a meeting directly to your task list
- **Authentication** — JWT-based auth with email/password and Google OAuth (Passport.js)
- **Dark/Light mode** — theme toggle persisted across sessions

## Tech Stack

**Frontend**
- React 19 + Vite
- React Router v7
- Tailwind CSS v4
- Axios

**Backend**
- Node.js + Express 5
- MongoDB + Mongoose
- OpenAI SDK (GPT-4o)
- Passport.js (Google OAuth 2.0)
- JWT + bcryptjs

## Getting Started

### Prerequisites

- Node.js 18+
- MongoDB (local or Atlas)
- OpenAI API key
- Google OAuth credentials

### Backend setup

```bash
cd server
npm install
```

Create a `.env` file in `server/`:

```env
PORT=5000
MONGO_URI=your_mongodb_connection_string
JWT_SECRET=your_jwt_secret
OPENAI_API_KEY=your_openai_api_key
GOOGLE_CLIENT_ID=your_google_client_id
GOOGLE_CLIENT_SECRET=your_google_client_secret
CLIENT_URL=http://localhost:5173
```

```bash
npm run dev
```

### Frontend setup

```bash
cd client
npm install
```

Create a `.env` file in `client/`:

```env
VITE_API_URL=http://localhost:5000
```

```bash
npm run dev
```

The app will be running at `http://localhost:5173`.

## API Routes

| Method | Endpoint | Description |
|--------|----------|-------------|
| POST | `/api/auth/register` | Register with email/password |
| POST | `/api/auth/login` | Login with email/password |
| GET | `/api/auth/google` | Google OAuth login |
| GET | `/api/tasks` | Get all tasks |
| POST | `/api/tasks` | Create a task |
| PUT | `/api/tasks/:id` | Update a task |
| DELETE | `/api/tasks/:id` | Delete a task |
| GET | `/api/meetings` | Get all meetings |
| POST | `/api/meetings` | Create a meeting |
| POST | `/api/meetings/:id/summarize` | Summarize meeting with AI |
| POST | `/api/meetings/:meetingId/tasks/:taskIndex` | Add AI task to task list |
| DELETE | `/api/meetings/:id` | Delete a meeting |
